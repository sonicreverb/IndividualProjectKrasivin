HTML парсер

I. Функциональные требования:
1. Ввод HTML кода представлен двумя способами:
через ручной ввод HTML кода в файл input.txt и при помощи GET запроса (пользователь вводит ссылку на сайт,
HTML код которого необходимо получить).
Ввод списка запрещённых сайтов осуществляется при помощи текстового файла.
Вывод осуществляется в текстовый файл, структура файла:

Ссылки, встречающиеся в данном HTML коде:
ссылка_1 кол-во_повторений
ссылка_2 кол-во_повторений
...
Ссылки на изображения, встречающиеся в данном HTML коде:
ссылка_на_изб_1 кол-во повторений
ссылка_на_изб_2 кол-во повторений
...

2. Возможности:
Сканирование исходного массива данных HTML кода, сбор, анализ
и упорядочивание информации, фильтрация по заданным параметрам.
Конвертация полученных данных в формат, удобный для использования.

3. Меню:
Программа представлена визуальным интерфейсом на русском языке.
В верхней части центральной области главного окна находится блок для работы с парсером.

В верхней части этого блока находится поле ввода,
предназначенное для введения URL адресса сайта, с которым программа будет взаимодействовать.

Ниже располагаются кнопки:
"Начать парсинг по ссылке" - осуществляет процесс парсинга по введённой ссылке;
"Начать парсинг из файла" - осуществляет процесс парсинга из файла;
"Изменить HTML код" - редактирования файла input.txt, внесение пользователем HTML кода вручную;
"Изменить список запрещённых ссылок" - редактирвание файла prohibited_sites.txt,

В нижней части центральной области главного окна находится блок для работы с заменой ссылок.
Блок представлен двумя полями ввода "исходная ссылка" и "новая ссылка" и кнопкой "Заменить ссылку в HTML файле".

II. Технологии:
Python 3.7, загружаемые библиотеки requests, BeautifulSoup4, Tkinter, PIL.
